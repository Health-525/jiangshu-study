# jiangshu-study（课程学习仓库）

这个仓库用来把本学期各门课的 **笔记 / 课件 / 教材 / 代码** 按课程归档，方便复习与检索。

---

## 目录结构

每个课程文件夹统一分为三个子目录：

```
课程文件夹/
├── 课件/     # 老师PPT、讲义PDF
├── 笔记/     # 个人整理的学习笔记（.md）
├── 教材/     # 教材PDF
└── README.md # 课程说明
```
