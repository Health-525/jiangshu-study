# CS 英语速查（自整理）

目标：把你在**数据结构与算法 / 数值分析 / 多元统计**里最常见的英文表达，整理成可背可用的短表。

## 1) 高频动词（写题解/报告必用）
- formulate：把问题表述为……（We formulate the task as a binary classification problem.)
- derive：推导（We derive the update rule from ...)
- assume：假设（We assume i.i.d. samples.)
- estimate：估计（We estimate parameters via maximum likelihood.)
- optimize：优化（We optimize the objective with gradient descent.)
- approximate：近似（We approximate the integral by ...)
- converge：收敛（The algorithm converges in ... iterations.)
- evaluate：评估（We evaluate using accuracy/F1/AUC.)
- compare：对比（We compare XGBoost with logistic regression.)
- ablate：消融（We conduct an ablation study on feature groups.)

## 2) 常见名词（统计/ML）
- feature / target / label：特征 / 目标 / 标签
- training/validation/test split：训练/验证/测试划分
- baseline：基线
- overfitting / underfitting：过拟合 / 欠拟合
- regularization：正则化
- bias-variance tradeoff：偏差-方差权衡
- cross-validation：交叉验证
- metric：评价指标
- leakage：数据泄漏

## 3) 算法/数据结构常用表达
- time complexity / space complexity：时间/空间复杂度
- amortized：均摊（amortized O(1))
- worst-case / average-case：最坏/平均情况
- invariant：不变量
- recurrence：递推式
- proof sketch：证明思路

## 4) 数值分析常用表达
- stability：稳定性
- conditioning / ill-conditioned：条件性 / 病态
- error bound：误差界
- floating-point：浮点
- round-off error：舍入误差
- truncation error：截断误差

## 5) 题解/报告模板句（可直接套用）
- Problem statement: ...
- Data description: ...
- Method: We start with a simple baseline and then ...
- Results: The proposed method improves ... by ...
- Limitations: Our approach may fail when ...
- Future work: We plan to ...
