## 学习目标

- 了解 Python 关键字

## 一、基本语法

### 注释

```python
# 单行注释

"""
多行注释
使用三引号
"""
```

### 缩进

Python 用**缩进**表示代码块，缩进必须一致：

```python
if True:
    print("True")
else:
    print("False")
```

### 语句换行

```python
# 方式1：使用圆括号
str = ('Python 是一种面向对象、'
       '解释型计算机程序设计语言。')

# 方式2：使用反斜杠 \
str = 'Python 是一种面向对象、' \
      '解释型计算机程序设计语言。'

# 注意：在 [], {}, () 内部不需要换行符
total = ['item_one', 'item_two',
         'item_three', 'item_four']
```

## 二、变量和数据类型

### 变量赋值

```python
num1 = 100
num2 = 87
result = num1 + num2

# 多变量赋同一值
x = y = z = 1

# 多变量赋不同值
a, b = 1, 2

# 变量交换
x, y = y, x
```

### 数据类型分类

| 类型 | 说明 |
|------|------|
| 数字类型 | int / float / complex |
| 布尔类型 | True / False |
| 字符串类型 | str |
| 列表类型 | list |
| 元组类型 | tuple |
| 字典类型 | dict |
| 集合 | set |

## 三、标识符和关键字

### 标识符命名规则

- 由**字母、下划线、数字**组成
- **数字不能开头**
- **区分大小写**
- 不能使用关键字

### 命名建议

- 见名知意：`student_name` 而非 `sn`
- 驼峰式：`studentName`

### 查看关键字

```python
>>> help()
help> keywords   # 查看所有关键字
help> return     # 查看某关键字说明
help> quit
```

## 四、简单数值类型

### 整型 int

```python
0b10100   # 二进制
0o10111   # 八进制
64        # 十进制
0x14      # 十六进制
```

### 浮点型 float

```python
3.14
1.5e3    # 科学计数法 = 1500.0
2.5E-2   # = 0.025
```

### 布尔型 bool

- 只有 `True` 和 `False`
- 以下值的布尔值均为 `False`：

| 值 | 说明 |
|----|------|
| `None` | 空值 |
| `False` | 布尔假 |
| `0` | 整型零 |
| `0.0` | 浮点零 |
| `""` | 空字符串 |
| `[]` | 空列表 |
| `()` | 空元组 |
| `{}` | 空字典 |

### 复数 complex

```python
5 + 3j   # real + imagj
```

### 类型转换

```python
int(x)           # 转整型
float(x)         # 转浮点型
complex(real, imag)  # 创建复数
```

## 五、运算符

### 算术运算符

| 运算符 | 说明 | 示例 |
|--------|------|------|
| `+` | 加 | `3 + 2 = 5` |
| `-` | 减 | `3 - 2 = 1` |
| `*` | 乘 | `3 * 2 = 6` |
| `/` | 除 | `3 / 2 = 1.5` |
| `%` | 取余 | `3 % 2 = 1` |
| `**` | 幂 | `3 ** 2 = 9` |
| `//` | 取整除 | `3 // 2 = 1` |

### 赋值运算符

```python
x = 1           # 基本赋值
x += 1          # 等效 x = x + 1
x -= 1          # 等效 x = x - 1
x *= 2          # 等效 x = x * 2
x /= 2          # 等效 x = x / 2
x %= 2          # 等效 x = x % 2
x **= 2         # 等效 x = x ** 2
x //= 2         # 等效 x = x // 2
```

### 比较运算符

| 运算符 | 说明 |
|--------|------|
| `==` | 等于 |
| `!=` | 不等于 |
| `>` | 大于 |
| `<` | 小于 |
| `>=` | 大于等于 |
| `<=` | 小于等于 |

### 逻辑运算符

| 运算符 | 说明 |
|--------|------|
| `and` | 两者都为 True 才返回 True |
| `or` | 有一个为 True 即返回 True |
| `not` | 取反 |

### 成员运算符

```python
'a' in ['a', 'b']      # True
'c' not in ['a', 'b']  # True
```

## 六、位运算

| 运算符 | 说明 |
|--------|------|
| `&` | 按位与：两位都为1才为1 |
| `\|` | 按位或：有一位为1就为1 |
| `^` | 按位异或：不同为1，相同为0 |
| `~` | 按位取反：0变1，1变0 |
| `<<` | 左移：高位丢弃，低位补0 |
| `>>` | 右移：移出位丢弃，补符号位 |

---

## 七、运算符优先级

> 括号优先级最高，建议复杂表达式**用括号明确优先级**

```python
# 假设 a=20, b=10, c=15, d=5
e = (a + b) * c / d   # = 90
e = a + (b * c) / d   # = 50
```

优先级从高到低：`**` > `~` > `* / // %` > `+ -` > 位运算 > 比较 > 逻辑
